## 👋 About Me

Hi, I'm Sameer! I'm a developer who loves working with cloud infrastructure, automation, and Linux system administration. This repo is a collection of the assignments I've completed during my learning journey — each one a real challenge I worked through from scratch.

## 🛠️ How I Did It

I approach every assignment with a clear workflow:

1. **Understand the problem** — what's the goal? (host a site, secure access, automate a deploy)
2. **Design the solution** — which services, tools, and configs do I need?
3. **Build and test** — write the code, run it, verify it works.
4. **Document** — explain what I did and why, so others can follow along.

For the bigger projects (like the Ansible WordPress workflow and the System Health Monitor), I also focused on making things reproducible and maintainable — reusable roles, modular configs, and clear documentation.
